# Menu Item Images

- `idly.jpg` - Image for Idly
- `dosa.jpg` - Image for Dosa
- `poori.jpg` - Image for Poori
- `vada.jpg` - Image for Vada
- `tea.jpg` - Image for Tea
- `coffee.jpg` - Image for Coffee
- `milk.jpg` - Image for Milk
- `boost.jpg` - Image for Boost

**Note:** If images are not found, the website will automatically use placeholder images from a CDN service.

Supported formats: JPG, PNG, GIF, WebP
